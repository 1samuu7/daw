# Proyecto Intermodular: InundaGo

InundaGo es una aplicación multiplataforma cuyo objetivo es ayudar a las personas a desplazarse de forma más segura cuando existen lluvias intensas o riesgo de inundación. La aplicación permite buscar rutas seguras, consultar información meteorológica y visualizar zonas peligrosas durante el trayecto.

## Funcionalidades principales
- Búsqueda de rutas completas.
- Visualización de zonas inundables.
- Información meteorológica actualizada.
- Gestión de usuarios y rutas favoritas.
- Interfaz clara y multiplataforma.

## Plan de despliegue
1. Clonar el repositorio del proyecto.
2. Instalar dependencias del proyecto.
3. Compilar el proyecto para generar los archivos finales.
4. Subir los archivos compilados a un servidor web.
5. Realizar pruebas para comprobar que el mapa, las rutas y las alertas funcionan correctamente.
